Overview
========

Keeping track of the configurations and settings I prefer in my user profile.


Usage/Installation
------------------
```
git clone https://github.com/sabrsorensen/dotfiles.git ~/dotfiles 
cd ~/dotfiles
git submodule update --init 
chmod +x dotfiles_setup.py
./dotfiles_setup.py
```
